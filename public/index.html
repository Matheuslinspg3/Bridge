Arquitetura

Duas camadas, uma solução

O Bridge opera em duas camadas complementares que entregam resiliência técnica e gestão de acesso.

⚡ Bridge Técnico

Proxy inteligente entre seu app e a API Claude.

  • Retry infinito com backoff exponencial
  • Throttling e concurrency control
  • Circuit breaker automático
  • Compatível OpenAI e Anthropic
  • Forced non-stream em overload

🔑 Portal SaaS

Gestão de assinaturas e API keys.

  • Cadastro e login com JWT
  • Planos com limite de tokens/mês
  • Pagamento via PIX (QR Code)
  • Rate limiting por chave (RPM + budget)
  • Dashboard de uso em tempo real
